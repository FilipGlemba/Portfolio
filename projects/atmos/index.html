Case Study

How this portfolio piece was shaped

Atmos was rebuilt from a simple weather lookup into a portfolio-grade weather dashboard focused on visual polish, product thinking, and safer architecture.

Challenge

Turn a basic demo into a product-like experience

The original version handled one API call and a single weather card. The goal was to make it feel premium enough for a modern developer portfolio.

Solution

Upgrade architecture, UX, and visual identity together

The app now uses a backend proxy, custom SVG weather icons, autocomplete search, geolocation, forecast views, and theme plus unit switching.

Stack

Vanilla frontend with a lightweight Node proxy

HTML, CSS, and JavaScript power the UI, while a minimal Node server protects the API key and proxies weather, forecast, geocoding, and air quality requests.

UX Focus

Fast feedback, strong readability, and memorable styling

The interface emphasizes clear hierarchy, responsive layouts, keyboard-friendly search, and subtle motion that improves feel without becoming noisy.
